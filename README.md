# resistance
Анализ сопротивления тонких пленок
В проекте представлен анализ сопротивлений тонких пленок, полученных в ходе технологического процесса:
1. Загружены и обработаны данные из xls файла. Проанализирована статистика сопротивлений.
2. Проведен А/А-тест образцов, сформированных в холодное и теплое время года.
3. Построены матрицы корреляций.
4. Произведена кластеризация образцов по величине сопротивления.

По результатам А/А-теста установлено, что пока нет достаточно сильных оснований утверждать, что различия в сопротивлениях образцов имеют статистическую значимость. Из матриц корреляций были установлены параметры процесса, с которыми наиболее сильно скоррелированы величины сопротивлений. Первоначальная кластеризация образцов по величине сопротивлений может быть уточнена в будущем.
